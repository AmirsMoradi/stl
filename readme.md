# Solid STL Viewer

Solid STL Viewer is a lightweight, semi-professional desktop application for viewing and inspecting STL files exported from CAD software such as SolidWorks.  
The application is built with **Python**, **VTK**, and **PySide6 (Qt)** and is designed to provide a clean, industrial-style user experience similar to professional CAD viewers.

---

## Features

- Open and visualize STL files
- Smooth rotate, zoom, and pan navigation
- Solid and wireframe rendering modes
- Change model color
- Distance measurement tool
- Section plane (clipping) for cross-sectional inspection
- Modern Qt-based user interface
- Standalone Windows executable (no Python installation required)
